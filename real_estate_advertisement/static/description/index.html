Property Sale & Rental Management

Odoo Real Estate Property Rental and Sale Management App

Few Words About The Property Sale & Rental Management

Property Sale & Rental Management app is complete solution all kind of property and housing Management system. We can create main properties and properties and set various type of details for a property. We can also create sale and rent offers for the properties and give our customers best price. We can create manage contract for rent and selling of property-house. Based on contract invoice will be created for every month according to the contract duration. This odoo apps allow partial payment for property selling, means we can accept payments in installments. In the configuration we create installment schemes. Once invoice is created invoice due date automatically set based on interval set on contract, and it applies fine after the payment due date.

Main Features
  • Create multiple properties of different types
  • Support Odoo default activity types
  • Create offers of rent and sale properties, also we can create offers for customers specifically
  • Create customized installment schemes for payment, Apply custom interest rates, and payment delay fine percent
  • Create contract for property sale and rent
  • We can select payment ways, Payment can be paid completely or paid in installments.
  • Payment reminder for delayed and upcoming payments
  • Send email to customer
  • Support multiple user group
